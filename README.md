## üìã List of Programs

### Arrays
1. **ArrayInsertDelete.java** - Insert and delete elements in 1-D array
2. **ReverseArray.java** - Reverse an array
3. **MatrixOperations.java** - Addition and multiplication of 2D matrices
4. **MaxMinArray.java** - Find maximum and minimum in array

### Recursion
5. **Factorial.java** - Factorial using iteration and recursion
6. **Fibonacci.java** - Fibonacci series using iteration and recursion
7. **TowerOfHanoi.java** - Tower of Hanoi problem
8. **PrintNto1.java** - Print numbers from n to 1 using recursion

### Searching
9. **LinearSearch2D.java** - Linear search in 2D matrix
10. **BinarySearch.java** - Binary search in sorted array
11. **FirstLastOccurrence.java** - Find first and last occurrence in sorted array

### Sorting
12. **BubbleSort.java** - Bubble sort implementation
13. **SelectionSort.java** - Selection sort implementation
14. **InsertionSort.java** - Insertion sort implementation
15. **QuickSort.java** - Quick sort implementation
16. **MergeSort.java** - Merge sort implementation

### Linked Lists
17. **SinglyLinkedList.java** - Menu-driven singly linked list with insert, delete, search, display
18. **DoublyLinkedList.java** - Doubly linked list implementation
19. **CircularLinkedList.java** - Circular linked list implementation

### Stack & Queue
20. **StackArray.java** - Stack operations (push, pop, isEmpty, isFull) using array
21. **CircularQueue.java** - Circular queue operations using array

### Trees
22. **BinaryTree.java** - Binary tree traversals (inorder, preorder, postorder)
23. **BST.java** - Binary Search Tree with insert, delete, search operations

### Graphs
24. **BFS.java** - Breadth-First Search traversal
25. **DFS.java** - Depth-First Search traversal
